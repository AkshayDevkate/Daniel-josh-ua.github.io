Celonis

Celonis provides an comprehensive approach for process enhancement which includes a Python-based Machine Learning Workbench for predictive insights, an AI- powered Action Engine for intelligent process recommendations and Process Automation to automate workflows. Celonis provides a highly scalable and secure platform, offers various deployment options and supports many databases and systems for data extraction. Celonis recently launched Operational Applications, which are role-based applications that automate tasks, prioritize workflows, and prescribe guidance against target business goals leveraging business context, AI, and the core process mining engine. The vendor is closely cooperating with universities through its Academic Alliance program.

Pro's

• Provide big picture information and drill down to details, such as actors, cases, documents
• Combine process discovery data with other business, e.g. controlling data
• Great user interface
• Great at being able to directly enrich the findings with cost or revenue data
• Easy to build and calculate critical process KPIs and demonstrate the findings to management (decision makers)

Con's

• It works with SAP HANA. Unfortunately, connecting it to the older SAP versions is a tedious process
• Working with data models is not easy. The data model has to be 100% correct, otherwise the results bring confusion (either visible or not visible) or the model does not provide any data
• Does not provide Support for all standard SAP modules
